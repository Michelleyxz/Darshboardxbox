Dashboard de Vendas
Este projeto consiste em um dashboard de vendas criado para visualizar, organizar e analisar dados de desempenho comercial de forma clara e objetiva.

Objetivo
Facilitar a análise de vendas, auxiliando na tomada de decisões por meio de gráficos e indicadores visuais.

Funcionalidades
Visualização de dados de vendas
Análise de desempenho
Organização clara das informações

Tecnologias Utilizadas
Python
Pandas
Matplotlib / Power BI / Excel (ajuste conforme o seu projeto)
